# Flashcard Quiz App 📚

A modern, interactive Flutter Flashcard Quiz application designed to help users learn and revise concepts efficiently using active recall and spaced repetition principles.

---

## 🚀 Features

* 🃏 Interactive flashcards (Question / Answer flip animation)
* ✨ Smooth 3D card flip transitions
* ➕ Add new flashcards dynamically
* ✏️ Edit existing flashcards
* 🗑️ Delete flashcards with confirmation dialog
* 🎨 Modern dark UI with glowing UI effects
* 📱 Fully responsive design (works on different screen sizes)
* 🔄 Navigation between cards (Next / Previous)
* ⚡ Lightweight and fast performance

---

## 🛠️ Tech Stack

* Flutter
* Dart
* Material Design
* Custom animations (AnimationController, Transform)

---

## 📸 UI Overview

* Dark futuristic theme
* Animated background particles
* Glassmorphism-style dialogs
* Gradient buttons and glowing accents

## ▶️ Getting Started

### Prerequisites

* Flutter SDK installed
* Dart SDK
* Android Studio / VS Code

### Installation

```bash
# Clone the repository
git clone https://github.com/Mariamawny/CodeAlpha_FlashCard.git

# Navigate to project folder
cd CodeAlpha_FlashCard

# Get dependencies
flutter pub get

# Run the app
flutter run
```
